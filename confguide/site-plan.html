Content

Home page

This is a "how to" guide on using conference rooms within the office building. Most of the conference rooms have a video conferecing and screen sharing presentation system call Teams Rooms. Below is a listing of conference rooms along with recommended capacitiies and scheduling guides. Also, there is a link on each page to a Teams Rooms step-by-step user guide to help you get familiar with the system interface and usage options. Further down the page is a building plan showing the locations of the conference rooms and the contact information to the IT Help Desk.
